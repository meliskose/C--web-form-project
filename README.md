# Yazılım Şirketi Web Form Uygulaması

## Proje Hakkında

Bu proje, Görsel Tabanlı Programlama dersi kapsamında C# Windows Forms kullanılarak geliştirilmiştir.

Uygulama, bir yazılım şirketinin müşterileriyle iletişim kurmasını ve müşterilerden teklif talepleri almasını amaçlayan form tabanlı bir sistemdir. Kullanıcılar şirket hakkında bilgi görüntüleyebilir ve teklif talep formu üzerinden gerekli bilgileri girerek başvuruda bulunabilir.

## Kullanılan Teknolojiler

* C#
* Windows Forms
* .NET Framework
* Visual Studio

## Özellikler

* Şirket tanıtım ekranı
* Teklif talep formu
* Kullanıcı bilgi girişi
* Form doğrulama işlemleri
* Kullanıcı dostu arayüz

## Kurulum

1. Projeyi bilgisayarınıza indirin.
2. Visual Studio ile çözüm dosyasını açın.
3. Projeyi derleyin ve çalıştırın.
4. Uygulamayı kullanmaya başlayın.
